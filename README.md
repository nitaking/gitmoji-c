gitmoji-c
=========

gitmoji-c is a [gitmoji-cli](https://github.com/carloscuesta/gitmoji-cli) clone tool that does not use gpg.

# Install

```
$ yarn global add gitmoji-c
```

# Usage

```
$ gitmoji --help
```

## Commit

```
$ gitmoji commit
```

## List

```
$ gitmoji list
```

## Search

```
$ gitmoji list <search word>
```

ex:

```
$ gitmoji list apple

üçé - :apple: - Fixing something on macOS.
üçè - :green_apple: - Fixing something on iOS.
```
